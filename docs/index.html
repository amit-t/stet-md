STET.MD · @amit-t/stet-md

Markdown review that stays in the file.

stet-md opens one Markdown file in a loopback browser UI, lets humans add threaded review comments, saves those threads back into the same Markdown file by byte splices, and lets AI agents reply through a safe CLI. Local-only. No telemetry. No server in the cloud.

Open full README

Stet.md is a single-file Markdown reviewer. Point it at a .md file and it serves a hardened review UI on 127.0.0.1. Comments are stored inline as structured stet:thread HTML-comment markers plus a generated visible blockquote — the Markdown file remains the source of truth.

Binaries: stet-md, stmd. Requires Node ≥20.10.

Local-first Byte-splice writes Loopback UI Agent CLI No telemetry MIT
